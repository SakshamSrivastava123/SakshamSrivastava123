📌 Project Overview
1. Customer Management System (CMS)
Customer Management System (CMS) is a full-stack web application that allows organizations to manage their customer data efficiently. The system features secure authentication, role-based authorization, and supports dual customer types: Individual & Corporate.

Users can add, edit, and soft-delete customers while maintaining data integrity and controlled access based on roles.
